# Fichiers publics du front e-depot

Déposez ici le **logo officiel MEDLOG** sous le nom exact :

    medlog-logo.png

Il est utilisé automatiquement sur toutes les pages (barre latérale, connexion,
inscription, activation), servi à l'URL `/e-depot/medlog-logo.png`.

Tant que le fichier est absent, l'application affiche le logo texte de repli
(aucun rendu cassé). Format conseillé : PNG à fond transparent, hauteur ≥ 88 px.
